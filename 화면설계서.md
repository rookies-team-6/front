## 화면 구조 설계

### 레이아웃 구성
```
┌──────────────────────────────┐
│         Header               │
├──────────────────────────────┤
│        Main Content          │
├──────────────────────────────┤
│         Footer               │
└──────────────────────────────┘

```

### 주요 섹션
| 섹션명 | 위치 | 역할 | 필수여부 |
|--------|------|------|----------|
| [Header] | 상단 | 로고(홈으로 이동), 로그아웃, 이용자 정보 확인, ▼ 버튼(드롭다운) | 필수 |
| [Header(Scorebar)] | 상단 | 조별 점수 확인(각 조별 페이지 이동 가능), 전체 답변 조회 | 필수 |
| [Content Area] | 중앙 | 컴포넌트 랜더링 | 필수 |
| [Footer] | 하단 | 단순 디자인 | 선택 |
